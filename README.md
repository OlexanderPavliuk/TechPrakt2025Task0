Виконання лабораторної роботи передбачає вивчення прийомів роботи з системою контролю версій **Git**:

- клонування віддаленого репозиторію;
- виконання комітів;
- створення гілок та перемикання між ними;
- злиття комітів у різних гілках;
- аутентифікація на **GitHub** за допомогою токена та через **SSH**;
- вивантаження віддаленого репозиторію на **GitHub**.

У __практичній роботі №1__ було:
1. створено локальний репозиторій
2. створено віддалений репозиторій на GitHub
3. зроблено коміти у локальному репозиторії
4. вивантажено усі коміти з локального у віддалений репозиторій

## [Використані команди Git](https://git-scm.com/doc)

У практичній роботі №1 було використано такі команди Git:

- [x] git init
- [x] git status
- [x] git add
- [x] git commit
- [x] git remote add
- [x] git push
- [x] git pull
- [x] git branch

Документація з використання Git розміщена за [посиланням](https://git-scm.com/doc)
